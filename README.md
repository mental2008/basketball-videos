# basketball-videos

这个仓库用于记录包括 NBA、NCAA 在内各大篮球联赛的比赛录像，均为互联网上公开的资源，方便大家查找、观看。

1. 如果内容涉及版权纠纷，可以联系我删除；
2. 描述中有任何错误或遇到链接失效的情况，欢迎提 Issue 或者 PR 进行修正；
3. 欢迎提 PR 增加更多的比赛录像。
